## 特点

- 🎨 沉浸式滚动叙事体验
- ✨ 粒子特效 + 电影蒙太奇开场
- 📱 完全响应式，移动端适配
- 🎵 支持 BGM 背景音乐
- ⚡ 纯静态，零依赖后端

## 本地运行

直接打开 `index.html` 即可，或使用任意静态服务器：

```bash
npx serve .
```
